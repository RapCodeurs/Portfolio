Site web pour un restaurant

J'ai réalisé ce projet pour un client qui voulait se lancer dans la restauration, et ouvrir son propre
restaurant.

Le but était de mettre sur pied son site web, qui devait mettre à la disposition des utilisateurs
une variétés de plats par son menu.

Nouveau dans les nouvellles technoligies, mon client voulait ube interface utilisateurs permettant déjà de choisir
menu mais aussi choisir un menu personnalisé. Alors je l'ai réalisé, comme vous le voyez
le site est fonctionnel et opérationnel.

Ce projet simple mais fonctionnel a donné satisfaction à mon client.

Technoligies utilisées: HTML, CSS, DJANGO, PYTHON, HEROKU
